Setup

Requirements

To go through this tutorial, you need the following software installed:

Python ≥3.5

Snakemake ≥5.24.1

Singularity ≥3.7.3

Plus

To follow this tutorial, a plus would be to have this basic knowledge and resources:

A working account on Tabebuia

Bases in BASH

Bases in Python

A working account on a cluster
